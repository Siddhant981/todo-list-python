# CLI To-Do List App

## Description
A simple command-line based To-Do List application built in Python. This app allows users to manage their tasks through a menu-driven interface. Tasks are stored persistently in a text file named `TodoList.txt`. The application displays the current date and time when viewing tasks.

## Features
- **Add Task**: Add new tasks to the list.
- **View Tasks**: Display all tasks with their line numbers and a current timestamp.
- **Remove Task**: Delete a specific task by entering its line number.
- **Menu-Driven Interface**: Easy-to-use menu for navigation.
- **Persistent Storage**: Tasks are saved to a file and persist between runs.

## Technologies Used
- **Python**: Core language for the application.
- **File Handling**: Uses Python's built-in file I/O for storing tasks.
- **Datetime Module**: For generating timestamps when viewing tasks.

## How to Run the Project
1. Ensure you have Python installed on your system (version 3.x recommended).
2. Download or clone the repository.
3. Navigate to the project directory in your terminal.
4. Run the application using the command:
   ```
   python todo_list_app.py
   ```
5. Follow the on-screen menu to add, view, or remove tasks. Enter '4' to exit.
